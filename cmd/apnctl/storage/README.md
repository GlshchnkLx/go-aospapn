# apnctl Example Storage

This directory contains small, deterministic APN datasets used by
`cmd/apnctl/README.md` examples.

- `apns-full-conf.xml`: compact AOSP-like source file.
- `vendor-apns.json`: standalone vendor source used for JSON-to-XML examples.
- `vendor.json`: curated JSON patch file with missing/new records.
- `vendor.xml`: the same kind of patch data in XML form.
- `ru-missing-defaults.json`: minimal patch file for a missing default APN.
- `out/`: scratch directory used by README commands.
